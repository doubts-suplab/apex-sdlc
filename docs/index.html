Step 2 — Quick start
Developer Onboarding
Angular · Java · Spring Boot · Mainframe · Estimated time: 20 minutes
Your AI Tools
Claude Code CLI GitHub Copilot IntelliJ / VS Code
ℹ️ You don't need to learn new tools. AI runs inside your existing IDE and terminal.
5 min
1
Install Claude Code CLI
One npm install. Works in any terminal — iTerm, VS Code integrated, IntelliJ terminal. Verify with claude --version.
$ npm install -g @anthropic-ai/claude-code
$ claude --version
# Sign in with your Anthropic team account
$ claude auth login
5 min
2
Open your repo & explore
Navigate to your Spring Boot or Angular repo. Run claude to start. Your CLAUDE.md context file is already there — Claude knows your stack.
$ cd payments-service
$ claude
# Try your first prompt:
"Explain this codebase architecture"
5 min
3
Enable Copilot in IDE
Install the GitHub Copilot extension in IntelliJ IDEA or VS Code. Sign in with your GitHub account (your seat is pre-provisioned).
# IntelliJ: Preferences → Plugins
# Search: "GitHub Copilot" → Install
# VS Code: Extensions (Ctrl+Shift+X)
# Search: "GitHub Copilot" → Install
3 min
4
Generate your first tests
Ask Claude Code to write JUnit 5 tests for any class. Review the output — accept, modify, or reject. Never commit without reading it.
"Generate JUnit 5 + Mockito tests for
all public methods in OrderService.
Use Testcontainers for DB calls."
2 min
5
Try Copilot inline
Open any Java or Angular file. Start typing a method. Copilot will suggest completions — press Tab to accept, Esc to dismiss. That's it.
// Type this in your Java service:
public List<Order> getOrdersBy
// Copilot completes → Tab to accept
∞
6
Share & learn daily
Post what worked (and what didn't) in #ai-sdlc on Slack. Browse the Prompt Library for battle-tested prompts from your peers. Tune your CLAUDE.md as you learn.
Golden rule: AI is a junior pair programmer. You're the senior. Always review before you commit.
