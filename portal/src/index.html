AI Usage Controls

Built-in guardrails to keep AI-generated work safe, auditable, and compliant

🔍
PII Guard
Two-layer scanner (regex + AWS Comprehend) scrubs sensitive data before any external AI call
⚙️
CI/CD Gates
GitHub Actions enforce golden rules, OWASP checks, and coverage gates on every AI-assisted PR
🚦
Mainframe Gate
Phase-controlled policy — analysis only until ARB approves code generation (Week 17)
📋
AI Review Board
Fortnightly ARB reviews violations, approves new tools, and signs off gate transitions
ℹ️

Mainframe Gate: Phase 0 — Analysis Only

AI may explain and document COBOL/JCL programs but must NOT generate or modify mainframe code until the ARB approves Phase 1 at APEX Week 17. See governance/policies/mainframe-gate-policy.md for the full gate criteria.
